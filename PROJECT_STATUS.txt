⚡ БЫСТРЫЙ СТАРТ НА RAILWAY

1. Создайте GitHub репозиторий и залейте код
2. Откройте railway.app → New Project → Deploy from GitHub
3. Подключите репозиторий
4. Добавьте PostgreSQL плагин (+ New → PostgreSQL)
5. Добавьте переменные окружения:
   - TELEGRAM_BOT_TOKEN
   - TELEGRAM_BOT_USERNAME
   - CLOUDINARY_* (уже заполнены)
   - NODE_ENV=production
6. Railway автоматически создаст APP_URL
7. Дождитесь build completion
8. Откройте публичный URL → готово!
